This file enables AI agents to discover and use the Brave Search API skill.

## Setup

Get API key: https://api.search.brave.com

Set `BRAVE_SEARCH_API_KEY` — see [brave-search/references/api-key-setup.md](brave-search/references/api-key-setup.md)

## Usage

Read the SKILL.md file from `brave-search/SKILL.md` for full instructions. Detailed endpoint documentation is in `brave-search/references/`.
